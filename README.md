## 기록 원칙

- 기록합니다: 소스 코드, 노트북, 분석 메모, 실험 로그, 작은 메타데이터, 제출 요약.
- 기록하지 않습니다: 원본 Kaggle 데이터, 큰 모델 파일, ONNX 파일, zip 제출 파일, 인증 정보, 로컬 개발 환경.
- 큰 산출물이 중요할 때는 파일 자체를 커밋하지 않고, 생성 방법과 제출 위치를 문서로 남깁니다.
